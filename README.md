# 🛡️ JiLinDNS

**JiLinDNS** 是一款专为极致网络体验设计的自建私人 DNS 服务。它集成了强大的广告拦截、隐私保护与全球访问加速功能，旨在为你提供一个更干净、更快速且更安全的互联网接入环境。

---

## ✨ 核心特性

* **🚫 强力拦截**：内置多重过滤黑名单，精准拦截网页广告、视频片头广告及第三方跟踪器（Trackers）。
* **⚡ 极速响应**：基于高效缓存机制，大幅降低常用域名的解析延迟，实现毫秒级响应。
* **🔒 隐私加固**：原生支持加密传输协议，有效防止 ISP 劫持、DNS 污染及上网行为监听。
* **🚀 网站加速**：优化主流网站及 CDN 节点的解析路径，显著提升网页加载速度。

## 🚀 快速接入 (Quick Start)

请根据你的设备类型或需求，选择以下任意一种方式接入 **JiLinDNS**：

### 1. 路由器全局设置 (IPv4)
适合家庭或办公环境全量设备接入。请在路由器的 WAN 口或 LAN 口设置中修改 DNS 地址：

| 优先级 | DNS 地址 | 说明 |
| :--- | :--- | :--- |
| **首选 DNS** | `103.233.254.183` | 请替换为你的服务器公网 IP |


### 2. DNS-over-HTTPS (DoH)
适用于 Chrome、Edge 浏览器、iOS、Android 13+ 以及各种现代化终端。具有极高的安全性和隐蔽性。

* **配置 URL**: 
  `https://dns.625674.xyz/dns-query`

> **设置示例 (Chrome)**: `设置` -> `隐私和安全` -> `安全` -> `使用安全 DNS` -> `选择自定义` -> 输入上方 URL。

### 3. DNS-over-TLS (DoT)
适用于 Android 系统原生私人 DNS 设置、高级防火墙及支持加密 DNS 的路由器。

* **主机名 (Hostname)**: 
  `//dns.625674.xyz`

> **设置示例 (Android)**: `设置` -> `连接与共享` -> `私人 DNS` -> `指定 DNS 域名` -> 输入上方主机名。
